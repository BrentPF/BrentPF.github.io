A 2D Platformer in Unity

A simple 2D platformer-style game made with the Unity Game Engine, along with its Unity Scripting API and C#. The game has no clear objective (currently), and is mainly for exploring the capabilities of the game engine.

Running and Jumping


Sprites in this game were acquired from Humble Bundle or were created by me. Here we can see the animation transitions and colliders also working well.

Attacking and Enemy Destruction


Hit colliders perfectly tuned for the bodies of the golems and the sword swing radius! The 2-part default attack for the MC also working nicely.
